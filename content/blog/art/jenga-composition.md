EDIT 1/12/12: Updated. The changes are subtle, but the CG model fits in with the rest of the image a bit better.

Here is my completed final project for my Shader Programming class. (WIPs have been moved to my scrapbook.) We needed to create a 3D model on the computer and then composite it into a real photograph to make it look real. Obviously, this jenga tower is a bit impossible, but I tried really hard to integrate it into the photo. (The colors look slightly different on different computer screens, but hopefully it still looks pretty good.) You can see the original that I used as a reference here to compare the two: [[link]](http://fav.me/d4ihfn3)
I'm happy with how this turned out, though it still feels kind of CG. I can't quite put my finger on why, but any further comments are, as always, welcome. ![:)](https://e.deviantart.net/emoticons/s/smile.gif)

Modeled in Maya 2012
Textured in Slim
Rendered with Renderman
Composited with Nuke
