I did some digging and found my original notes for the very first few pages of *Aww, Feathers!* You can see I included the Spanish translation alongside each English line. I still translate each page into Spanish, but now I save the translation until the very end, because only then is the dialog completely locked down; there can still be slight changes here and there until then. (The Spanish version of each page is not currently showing on the website, but that's on my backlog.)
